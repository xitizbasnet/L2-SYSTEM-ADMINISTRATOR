## ⭐ Best Practice Coverage Areas

Across all exercises, focus on:

* Active Directory administration & Group Policy control
* DNS, DHCP, and core TCP/IP troubleshooting
* Windows Server role configuration (IIS, File Services)
* Security hardening (firewall, permissions, port security, ARP protection)
* Automation using PowerShell & scheduled tasks
* ITIL workflows (incident, change, problem management)
* Enterprise ticketing systems (Jira, ServiceNow)
